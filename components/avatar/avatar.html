Avatar

Normal avatar

Avatar is applied on photos, icons and can be created by inseting img tag inside avatar class avatar is basic class which has width:10rem

Avatar Sizes

In sizes we have three types: avatar-xl extra large, avatar-md medium size, avatar-sm small size. Don't forget to put avatar class before these as it provides that round shape

Avatar square

avatar-sq provides square shape to avatar
